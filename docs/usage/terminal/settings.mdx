---
title: Settings
---

Default settings can be edited via a configuration file. To open the file, run:

```bash
interpreter --config
```

| Key                     | Value            |
|-------------------------|------------------|
| `llm.model`             | String ["openai/gpt-4", "openai/local", "azure/gpt-3.5"]|
| `llm.temperature`       | Float [0.0 -> 1.0]                 |
| `llm.supports_vision`   | Boolean [True/False]                 |
| `llm.supports_functions`| Boolean [True/False]                 |
| `llm.context_window`    | Integer [3000]                 |
| `llm.max_tokens`        | Integer [3000]                 |
| `llm.api_base`          | String ["http://ip_address:port/v1", "https://openai.com"]|
| `llm.api_key`           | String ["sk-Your-Key"]                 |
| `llm.api_version`       | String ["version-number"]                 |
| `llm.max_budget`        | Float [0.01] #USD $0.01                |
| `offline`               | Boolean [True/False]                 |
| `vision`                | Boolean [True/False]                 |
| `auto_run`              | Boolean [True/False]                 |
| `verbose`               | Boolean [True/False]                 |
